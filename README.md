This is a completed web project where you can keep track of all the places you've visited. Each place has properties like **location, landmarks, best season to visit, and personal notes**. Clicking on a place shows its details.

This project uses **JavaScript css,and html**.

---


## features
- Add a place with location, landmarks, season, and optional notes  
- View a list of all added places  
- it can delete data
- it can also edit the data

---
## How to use 
- Fill in the formn with location, landmarks, season,and notes
- click "save place" to add it
- click aplace in the list to view details
- click delete to remove the detail
- click edit to edit the the detail
